# 🔐 Random Password Generator

A simple and secure Random Password Generator built using Python.  
This project was developed as part of my internship task at Oasis Infobyte.

## 📌 Project Overview

This program generates a random password based on user input.  
The user can specify the number of:
- Letters
- Digits
- Symbols

The program then creates a unique and randomized password every time it runs.

---

## 🚀 Features

- Takes custom input from the user
- Generates a different password every time
- Uses Python's `string` module
- Uses Python's `random` module
- Implements `range()` and loops
- Includes basic error handling

---

## 🛠 Technologies Used

- Python 3
- string module
- random module
